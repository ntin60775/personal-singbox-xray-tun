# Архив исторических планов

Каталог `plans/` больше не используется как рабочий контур задач. Актуальный workflow перенесён в `knowledge/tasks/`, где для каждой задачи ведутся `task.md`, `plan.md` и запись в `knowledge/tasks/registry.md`.

## Что хранится здесь

- ранее созданные планы, оформленные до перехода на knowledge-систему;
- исторические roadmap и черновики, которые пока не перенесены в `knowledge/tasks/`;
- справочный контекст для адресной миграции старых материалов в отдельной задаче.

## Как обращаться с архивом

- не создавать здесь новые планы, черновики и roadmap;
- не переносить сюда новые task-артефакты;
- не переписывать старые файлы без отдельной задачи на миграцию или архивную корректировку;
- использовать материалы из `plans/` только как историческую справку или как источник для осознанной миграции.

## Структура архива

- `drafts/` — исторические черновики и отложенные идеи
- `active/` — исторические master-планы и дорожные карты, которые ещё не разложены на задачи knowledge-системы
- `done/` — завершённые исторические планы
- `cancelled/` — отменённые исторические планы

Структура каталогов сохраняется для совместимости и навигации. Каталоги не удаляются автоматически и остаются частью репозитория как архив.
