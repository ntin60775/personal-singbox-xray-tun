# Доработка Clash-кандидата по замечаниям со скриншота

- Дата: 2026-04-02
- Статус: done
- Источник: замечания пользователя по fullscreen Clash-кандидату на `/design-review`

## Цель

Исправить конкретные UX-проблемы текущего Clash-кандидата: убрать лишнюю верхнюю плашку, развести sidebar и основную рабочую область, уплотнить верхнюю панель и заменить пустое место полезной сводкой.

## Изменения

- Убрать верхнюю review-плашку, которая отъедает высоту и не несёт ценности внутри самого интерфейса.
- Увеличить визуальный зазор между sidebar и основной рабочей областью, чтобы панель не «наезжала» на боковое меню.
- Уменьшить высоту верхнего header-блока рабочей области.
- Добавить в верхний блок компактные полезные метрики вместо пустого пространства.
- Сохранить fullscreen-поведение без вертикального скролла для развёрнутого desktop-окна.

## Проверки

- `python3 -m py_compile gui/gui_server.py`
- `python3 -m unittest tests.test_gui_server`
- Локальный HTTP smoke:
  - открыть `/design-review`;
  - проверить, что верхняя review-плашка убрана;
  - проверить, что clash-кандидат остаётся основным единственным экраном.

## Допущения

- Замечание «это убрать» относится к верхней review-плашке над самим интерфейсом, а не к внутреннему sidebar-блоку бренда.
- Сценарий fullscreen по-прежнему приоритетнее, чем поведение в узком окне.

## Итог

- Верхняя review-плашка полностью удалена, чтобы не тратить вертикальную высоту на внешний служебный слой.
- Между sidebar и рабочей областью добавлен явный зазор через внутренний `gap` у основного окна и пересобранные отступы workspace.
- Верхний header рабочей области уменьшен по высоте и дополнен полезной сводкой:
  - рабочие узлы;
  - свежие подписки;
  - состояние стека.
- Fullscreen-режим для развёрнутого окна сохранён: основная поверхность по-прежнему собирается в высоту viewport без отдельной вертикальной ленты страницы.
- Подтверждено:
  - `python3 -m py_compile gui/gui_server.py`;
  - `python3 -m unittest tests.test_gui_server`;
  - локальный HTTP smoke на `/design-review` с проверкой новых верхних метрик.
