## Цель

Устранить критичный функциональный разрыв: TUI должен уметь активировать узел, импортировать подписку по URL и импортировать отдельные ссылки — без этих функций невозможен основной сценарий использования.

## Границы

### Входит

- Активация узла из таблицы (DataTable row action)
- Модальный диалог импорта подписки (Название + URL)
- Модальный диалог импорта ссылок (TextArea)
- Замена заглушек `_action_import_subscription` и `_action_add_manual`
- Подключение через `subvost_app_service.py`

### Не входит

- Ping, удаление подписок, управление routing (P1)
- Управление профилями, geodata (P2)
- Изменение Windows-части
