  - согласованность маршрутов и GUI asset.
- Подтверждено:
  - `python3 -m py_compile gui/gui_server.py gui/gui_contract.py`;
  - `bash -n libexec/open-subvost-gui.sh`;
  - `python3 -m unittest tests.test_gui_server`;
