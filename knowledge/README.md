# База знаний проекта

`knowledge/` — канонический корень проектной памяти.
Здесь хранится всё, что нужно для сопровождения задач, решений и связанных операционных действий.

## Разделы

- `tasks/` — задачи и подзадачи, внутри которых хранятся карточка, план, журнал, решения и артефакты.
- `operations/` — действия, которые не считаются задачами: сборка из исходников, служебные скрипты, операционные процедуры и техническое обслуживание.

## Точки входа

- `tasks/README.md` — правила хранения и ведения задач вокруг единого центра задачи.
- `tasks/registry.md` — навигационный реестр задач и подзадач.
- `tasks/_templates/` — шаблоны обязательных файлов задачи.
- `operations/README.md` — правила для операционных действий, не являющихся задачами.
